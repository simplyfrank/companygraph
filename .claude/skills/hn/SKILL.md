# Hacker News Digest

Browse top Hacker News stories and search.

## Telegram Commands

- `/hn` -- Top 10 stories
- `/hn <N>` -- Top N stories (max 30)
- `/hn search <query>` -- Search HN via Algolia API

## API

- Top stories: HN Firebase API (no key needed)
- Search: Algolia HN API (no key needed)

## After Running

Show stories with links, scores, and comment counts. Offer refresh or expand.
