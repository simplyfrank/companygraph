# Pomodoro Timer

Start focused work sessions with Telegram notifications when time's up.

## Telegram Commands

- `/pomodoro` -- Show timer status or start menu
- `/pomodoro <minutes> [task]` -- Start a pomodoro (default: 25 min)
- `/pomodoro stop` -- Stop the current session

## Features

- Configurable duration (default 25 min work, 5 min break)
- Auto-suggests break after work session
- Tracks completed pomodoros count
- Sends Telegram notification when timer expires
- Inline buttons: Start, Stop, Break, Extend (+5m)

## After Running

Show timer status or offer to start a new session.
