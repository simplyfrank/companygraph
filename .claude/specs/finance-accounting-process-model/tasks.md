# Tasks: finance-accounting-process-model

## Reading guide

- **Order**: tasks execute top-to-bottom. Dependencies are explicit
  (`Blocked by` / `Blocks`); no out-of-order execution.
- **Deferred-green rule**: every seed/link/dedup task drives the loopback API on
  `127.0.0.1:8787` and reads Neo4j/Postgres, so its **integration** test needs a
  running API + Neo4j + Postgres. At each such task's checkpoint run
  `bun run typecheck`; the `*.integration.test.ts` files run green under
  `bun test:integration` once the stack is up (`bun run dev`). This spec ships
  **no** `pwa/` code, so there is no `bun test` component slice and no
  `design-conformance` invocation.
- **Verification**: every task declares a concrete test path or
  `manual: <one-line repro with input mode + observable outcome>`. The completion
  hook blocks STATUS.md updates without one.
- **Complexity**: `simple` (≤30 min mechanical), `moderate` (≤2 h, one judgment
  call), `complex` (≤half-day, multiple judgment calls).
- **Validation checkpoint**: after every task, run `bun run typecheck`; after
  tasks that ship behaviour, also run the listed integration test.
- **Ownership guard (XD-04/XD-05/XD-08/NFR-01/NFR-03/NFR-06)**: this spec is
  **data-only**. No task edits any schema array, route file, view file,
  `route.ts`/`SURFACES`/`views/index.tsx`, or owned-elsewhere storage/route code.
  The complete owned-file set is: `shared/seed/saas-operator/finance-accounting.json`,
  `api/scripts/seed-finance-graph.ts`, `api/scripts/seed-finance-risks.ts`, the
  two `package.json` script lines, a self-owned `api/scripts/finance-ids.ts`
  constant module, and the five `api/__tests__/finance-*.integration.test.ts`
  files. **Off-limits (never edited):** `shared/src/schema/{nodes,edges}.ts`,
  `api/src/routes/{kpi-crud,stories,risk-register,compliance-rules,change-requests,sla-crud,import,edges,ddd-system,capabilities}.ts`,
  `api/src/storage/{model-lifecycle-guard,capabilities,system-model}.ts`,
  `api/src/seed/{ensure-operator-root,ensure-function-domains,ensure-catalog,link-kpi-metric,governed-seed-helper}.ts`,
  `api/scripts/seed-saas-operator.ts`, `api/src/auth/rbac-permissions.ts`,
  `api/src/errors.ts`, and all of `pwa/**`.
